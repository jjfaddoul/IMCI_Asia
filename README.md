# IMCI Asia - Software Portfolio

A GitHub Pages website showcasing IMCI Asia's portfolio of software projects.

## Overview

This is a responsive portfolio website built with HTML and CSS, designed to showcase innovative software solutions developed by IMCI Asia.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean and professional design with smooth animations
- **Project Showcase**: Grid layout displaying software projects with descriptions and tags
- **Easy Navigation**: Fixed header with smooth scroll navigation

## Sections

- **Hero**: Welcome section with call-to-action
- **About**: Information about IMCI Asia
- **Projects**: Grid of featured software projects
- **Contact**: Contact information and call-to-action

## Technologies

- HTML5
- CSS3 (with CSS Variables and Flexbox/Grid)
- Google Fonts (Inter)

## Local Development

To view the website locally, simply open `index.html` in a web browser.

## Deployment

This website is designed to be hosted on GitHub Pages. It will be automatically available at `https://[username].github.io/IMCI_Asia/` once GitHub Pages is enabled in the repository settings.
